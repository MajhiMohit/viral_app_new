## About This Project

**Go Viral** is a content analytics tool where creators upload videos or images and receive an AI-powered virality score (0–100) with detailed, actionable feedback.

Built using:
- React + Vite + TailwindCSS v4
- Framer Motion animations
- Anthropic Claude API (`claude-sonnet-4-20250514`)
- Zustand state management

## AI Tool Used

**Antigravity** by Google DeepMind — an AI coding assistant that generated all source files, components, API integrations, and design system through structured prompting.

Model: **Claude Sonnet 4.6 (Thinking)**
