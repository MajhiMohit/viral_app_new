## Features Built by AI

### 1. Upload System
- Drag-and-drop zone accepting MP4, MOV, JPG, PNG, WEBP
- Video preview with thumbnail extraction at 0s, 1s, 3s
- Image preview with zoom
- File size validation (50MB video / 10MB image)
- Progress indicator during upload

### 2. AI Virality Score Engine (0–100)
- Calls Claude API with base64 image input
- Returns structured JSON: virality_score, hook_score, pacing_score, thumbnail_score, caption_score, insights, rewrites, hashtags, audio, competitors
- Animated circular SVG progress ring + count-up number
- Confetti animation when score ≥ 85

### 3. Hook Analysis
- First 3-second hook scoring out of 100
- 3 AI-generated hook rewrite suggestions
- Hook formula tag (Question Hook, Shock Hook, Story Hook, etc.)

### 4. Caption Optimizer
- Side-by-side original vs AI-rewritten caption
- Emoji density, CTA strength, character count vs platform limits
- One-click copy for optimized version

### 5. Hashtag & Audio Recommendations
- 12–15 hashtags: Trending (red), Niche (blue), Evergreen (gray)
- Click-to-copy individual or all hashtags
- 3 trending audio suggestions with genre and vibe tags

### 6. Competitor Benchmarking
- Animated horizontal bar chart vs: Top 1% creators, Viral threshold, Niche average, Platform average

### 7. Detailed Metric Breakdown
- 4 cards: Hook Strength / Pacing & Rhythm / Thumbnail Appeal / Caption Power
- Animated score bars, insight text, specific fix per card

### 8. Bonus Features
- Re-analyze button with score diff view
- Export as PDF (jsPDF + html2canvas)
- History tab — last 5 analyzed posts in localStorage
- "Fix It For Me" — re-calls Claude with rewrite strategy intent
- Confetti at 85+ score (canvas-confetti)
